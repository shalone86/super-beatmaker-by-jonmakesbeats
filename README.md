# Super-Beatmaker by Jon Makes Beats
A beat-making roguelike: to help you make beats in a creative and challenging way.

Features:
- share your seed so other players can make a beat based on your run
- collaborate on a beatmaker run with other players by exporting/importing runs
- play anywhere on any device, simply download the HTML file and run it in a broswer

Please support the creator of the game at http://patreon.com/jonmakesbeats
